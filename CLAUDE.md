# NOCTURNE — Luxury Lingerie E-Commerce

Dark luxury lingerie brand, targeting Western markets (US/UK/EU).

## Tech Stack
- Next.js 14 (App Router) + TypeScript strict
- TailwindCSS v3 + shadcn/ui
- next-intl (EN/FR/DE/ES/IT)
- Prisma + Vercel Postgres (later)
- next-auth v5 (later)
- Stripe (later)

## Color Palette (WARM CHAMPAGNE — NOT white)

## Typography
- Headings: Cormorant Garamond (serif, Google Fonts)
- Body: Inter (sans-serif)
- CTAs: Montserrat (uppercase, tracked-out)

## Key Rules
- NO pure white (#FFF) or pure black (#000) anywhere
- All product names are women's first names (VIOLETTA, ELARA, etc.)
- Tone: sensual, editorial, empowering — NEVER vulgar
